Example CRUD RESTful web application with AngularJS front-end

Used technologies:

Server Side:

  • Spring MVC
  • Spring Data
  • Hibernate
  • HSQLDB
  • JUnit
  • Mockito
  • Maven

Client side:

  • AngularJS
  • Bower
  • Grunt

Default access the deployed web application at: http://localhost:8080/customers-database/

Default access the REST API at: http://localhost:8080/customers-database/api/customers
